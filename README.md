# PiPico
A collection of code and notes concerning my Raspberry Pi Pico devices

I ran these programs using Thonny 3.3.1 on a Raspberry Pi 400.

## Packages used
- **picozero**
- **ssd1306** - I think I use micropython-ssd1306


## Files

- **hello.py** - a simple Python program to test Thonny
- **blink.py** - a simple blink program using picozero to blink the onboard LED
- **blink15.py** - a blink program that blinks an external LED at pin 15
- **toggle.py** - a program that toggles the LED on and off if a button is pressed.
- **oledtest.py** - a program to write to an OLED display of 128x32 
- **oledtest2.py** - more advanced text

- Working on some new weather code to add.
